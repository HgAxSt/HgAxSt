## Projetos

### 🎙️ Arteus — Análise Vocal com IA

Ferramenta de **análise de performance vocal** com processamento de áudio de alta precisão.

| Componente | Tecnologia |
|---|---|
| Feature Extraction | Librosa — MFCCs, pitch, spectral centroid |
| Transcrição + Alinhamento | WhisperX |
| Frontend | Vue.js |
| Backend | Flask (REST API) |

Pipeline completo: do raw audio até o relatório de análise de afinação, timbre e dinâmica do cantor.

### 🎵 Classificação de Gêneros Musicais &nbsp; [![repo](https://img.shields.io/badge/repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/HgAxSt/Identifica-o-de-g-neros-musicais)
